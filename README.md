# FileAutoTraveler
A simple Windows program that moves detected files into starting folders and moves them into ending folders.
Each control loops in a separate thread so the moves can be performed in parallel.
You can also choose the specific format of the files you want to move.

## Important infos
Find the program in releases. You'll need to install the desktop runtime to run this application.
The release contains a ".bat" file that will redirect you directly to download the correct version of the required Microsoft .NET Desktop Runtime.
